About

A Python-based program that analyses vehicle movement at road junctions and generates histogram-based visualisations to support traffic pattern evaluation and management decisions.



Features

- 🚗 **Vehicle Movement Analysis** — Tracks and analyses traffic flow at road junctions
- 📊 **Histogram Display** — Generates clear visual histograms of traffic pattern data
- 🔄 **Traffic Management** — Supports evaluation of congestion and flow patterns
- 🐍 **Pure Python** — Built entirely with Python and its data science libraries
